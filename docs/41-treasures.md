# 寶箱 / 隱藏物品(城鎮 + 迷宮)— 自 CTY*.DAT 萃取

> 接續 docs/40 設施系統,把「每個寶箱/隱藏點裝什麼」也從地圖資料抽出來。
> 兩者同源:設施走 section header `+6`,寶箱走 `+8`(examine 事件表)。

## 一、資料路徑(docs/31/35 §三)

踩到/面向有事件的 tile 按 Enter →(地表 0x9cf6 / 城鎮 0x9f7f)讀 tile,`attr&8` 才有事件。
事件項由 tile 高 byte 低 5 bit(subid)索引 section 的 examine 事件表:

```
ev_tbl = section_base + word[section_base + 8]
ev_tbl[0] = count                       ; 表首 1 byte = entry 數
entry i   = ev_tbl + 1 + i*4            ; 每筆 4 byte
  type  = entry[0]                       ; 0=調査/寶箱 1/3=寶箱給道具 2=傳送 4=特殊
  param = entry[1..2] (u16)              ; type 0/1/3 → 道具 id(= ITEM.DAT index)
  flag  = entry[3]                       ; 一次性旗標 id([0x4f70] bit:set=未取、清=已取)
```

派發(0x9d86):type 0(0x9ec1)`param==0`→「在腳邊調查‥/可惜是空的」、`param!=0`→給道具;
type 1/3(0x9de9)→ 掃隊伍背包 8 格存入 param(滿了印「行李滿」);type 2(0x9f32)→ 傳送門。

## 二、萃取

`tools/gen_treasure_data.py` 讀 `CTY*.DAT` 逐 section 的 `+8` 表,收 type 0/1/3 且 param 為
有效道具 id 的項 → `dq3_remake/src/dq3_treasure.{c,h}`。**全 89 城共 121 寶箱/隱藏物品、橫跨 42 城。**

道具名 = D3TXT00 rec `(item_id+1)`。交叉驗證(皆與杜勝利攻略/劇情自洽):
- CTY13 迷宮 **黃金爪**(金字塔)、**魔法鑰匙**;CTY40 **最終鑰匙**;CTY55 **蓋亞之劍**。
- CTY80 **太陽之石** → 對上彩虹水滴合成材料(docs/31 #2);CTY90 **賢者之石** + 6×世界樹之葉(終盤)。
- 各色寶珠(紅/藍/紫/黃/夢幻紅寶石)散落對應城,符合「集寶珠」主線。

## 四、remake 落地

- `dq3_treasure.{c,h}`:`dq3_treasure` 陣列(cty/sec/sub/type/item/flag)+ `dq3_treasures_for(cty)`。
- `main.c` 城鎮 Enter「調べる」:面向格(其次本格)有 type 0/1/3 + 有效道具 → `dq3_inv_add` 入背包 +
  以 `dq3_flags_set(flag)` 標記已取(remake 語意:set=已取);取過再調べる回「可惜是空的」(rec 0xf3)、
  背包滿回「行李滿了」(rec 0x13a)。`dq3_scene_tile_event_p2` 新增回傳 p2 旗標供此用。
