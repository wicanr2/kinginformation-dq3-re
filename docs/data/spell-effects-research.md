## remake 實作決策(簡化戰鬥/世界模型下的忠實對應)

- **戰鬥狀態模型**:加 battle 修正狀態(敵/我 atk·def·agi 倍率 + 我方 blind/sealed flag)。怪施 base==0
  咒 → 套對應修正(取代 no-op 假傷害):史卡拉/史克魯多→敵 def↑、拜基魯多→敵 atk↑、皮歐里姆→敵 agi↑、
  魯卡尼/魯加南→我方 def↓、瑪努莎→我方 blind(物攻易 miss)、瑪荷頓→我方 sealed(不能施咒)、
  夫巴哈/阿斯特龍/反彈→記旗標(吐息/特殊未模型化者註記)。倍率採 DQ 經典(×2 攻、×0.5 守等),
  精確 stage 公式未逐一 RE → 用合理值 + 本表標註(同咒文威力表處置)。
- **野外咒文**:cmd「咒文」改為可施放;實作 魯拉(回出發城/地表起點)、烈米特(出城回地表)、
  托黑洛斯(驅弱敵,重用 REPEL)。其餘(因帕斯/拉那路達/帕爾奔得)記為後續。

# remake 實際實作對照(2026-06-27 落地)

> 本段說明 remake **怎麼做**每個咒文效果、對應 code、以及忠實度標註(精確 RE vs 經典近似)。

## 戰鬥狀態/輔助咒(`dq3_battlescene.c`)

機制:加 5 個 per-battle 修正狀態(`reset_battle_mods()` 每場歸零),怪施 base==0 咒時改之、
do_turn 套用。倍率用百分比整數,採 **DQ 經典語意**(精確 stage 公式未逐一 RE,同咒文威力表處置)。

| rec | 咒名 | remake 效果 | code |
|---|---|---|---|
| 144 | 拉里荷(睡)| `status \|= PARALYSIS`(不能行動)| do_turn 敵施咒段 |
| 152 | 美達巴尼(混亂)| 同上映射麻痺 | 同上 |
| 151 | 拜基魯多 | `g_eatk_pct *= 2`(上限 400)→ 敵物攻 ×2 | do_turn 入口 `eatk = eatk*g_eatk_pct/100` |
| 154/155 | 史卡拉/史克魯多 | `g_edef_pct += 50`(上限 300)→ 敵守備↑ | do_turn 入口 `edef = edef*g_edef_pct/100` |
| 146/147 | 魯卡尼/魯加南 | `g_pdef_pct -= 25`(下限 25)→ 我方守備↓ | 敵物攻 `party.def*g_pdef_pct/100` |
| 158 | 瑪努莎 | `g_party_blind=1` → 我方物攻 ~50% 失手(roll<128 揮空)| do_turn 戰鬥段 |
| 156 | 瑪荷頓 | `g_party_sealed=1` → 我方 cmd==4 施咒被擋 | do_turn 咒文段 |
| 其餘 base==0 | 皮歐里姆/夫巴哈/阿斯特龍/反彈/巴西魯拉/莫夏斯/尼夫拉姆/瑪荷多拉/伯米歐斯 | 詠唱**不致傷**(效果未模型化,誠實標註;非假傷害)| do_turn `nm==0` 分支 |

> 修掉的真 bug:前版這些 base==0 咒 `spell_def_get` 回 NULL → fall through 成 base=24 **假傷害**。
> 現在改成套修正狀態或詠唱不致傷。
