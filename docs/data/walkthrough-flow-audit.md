## 結論:三層缺口

> **重要前提**:這些**原版(底層 ENIX DQ3)全都有**(使用者親自玩過 DOS 版破關=ground truth)。
> 精訊**中文化是做完了的**——只是**有 bug、缺部分圖像**,不是核心遊戲內容缺。下面「未接」一律是
> 「**remake 尚未徹底反組譯 / 尚未接線**」,**不是**「原版沒這段 / build 沒接 / 中文化沒做完」。
> 反組譯難(real-mode/overlay/資料驅動 runner)是我們的成本,不是內容不存在。
> 杜勝利攻略**很準**,當原版 ground truth 逐段 RE。

1. **可玩性**:remake 用自有里程碑流**已可破關**(不靠這些)。
2. **忠實度最大缺口 = 巴拉摩斯**(下降前主線大 boss,remake 尚未接)+ 甘達特金皇冠正源。
3. **長尾**:關鍵道具取得鏈(夢幻紅寶石/變身杖/蓋亞之劍/鑰匙/彩虹材料)原版都有,
   remake 目前「道具能用但取得鏈尚未 RE 接線」;照杜勝利攻略逐段 RE 觸發 + 以 remake 旗標流補,
   即可串成「照攻略玩得通」。
