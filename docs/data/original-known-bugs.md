## ★ 金字塔內使用咒文 → 當機(使用者回報 2026-06-26)

- **現象**:原版一進入**金字塔(CTY13)**後,在裡面**使用咒文(野外/戰鬥?)就會當機**。
- **意義**:這是**原版固有 bug**(中文化做完但有 bug)。remake 不應復刻此 crash;若 remake 在金字塔
  施咒正常,即「修對了原版的 crash」(類似 7 主 bug 的處理態度)。
- **RE 待查(線索)**:
  - 金字塔 = CTY13(docs/maps/cty-names)。為何「該地圖 + 咒文」會 crash?
  - 假說:咒文選單/施放路徑(0xbfd0 effect dispatcher / 0xc22e 扣血,docs/13)在金字塔 section
    存取了某個未初始化/越界的地圖相關狀態(如 overlay 未載、某指標在該 CTY 為 0xffff)。
  - 或:金字塔有特殊機關(按鈕 G/H 開石門取魔法鑰匙),其 section 事件/旗標與咒文系統共用某全域
    被破壞。
  - 行動:之後在 DOSBox 原版於 CTY13 施咒重現 + 看 crash 位址,反推。**先記錄,不阻塞當前工作。**
