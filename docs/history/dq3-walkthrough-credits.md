## 杜勝利攻略 — 內容導覽(摘要,非原文)

> 著作權說明:以下為杜勝利攻略的**結構摘要與重點索引**,供本專案交叉佐證之用;
> **非逐字轉錄**。完整原文請見原始出處 `vv0817.neocities.org/gametxt/09_dq3`,著作權歸杜勝利。

杜勝利這份攻略把 DQ3(精訊版背景)從主角十六歲生日的阿里阿罕開場,一路寫到對戰大魔王索瑪,
重點章節與本專案的對應:

- **開場(阿里阿罕)**:主角生日 → 見國王 → 城外酒店挑同伴。對應 remake 開場(CTY0 酒場創角)。
- **盜賊鑰匙線**:阿里阿罕 / 雷貝南方洞窟 → 拿吉米之塔塔頂取盜賊鑰匙 → 雷貝取魔法球 → 誘人洞窟炸牆。
- **羅馬利亞線**:旅人之門 → 羅馬利亞(找皇冠)→ 卡薩布(毒針)/ 諾阿尼魯(催眠事件)/ 香巴尼塔(甘達特)。
- **沙漠線**:阿莎拉慕(拍賣場:理力之杖 / 鐵頭盔)→ 金字塔(魔法鑰匙 / 黃金爪)→ 依西斯(流星手環 / 祈禱戒指)。
- **取船後**:波魯多加(胡椒換船)→ 巴哈拉達 → 達瑪(轉職)→ 日邦格(八頭大蛇)…
- **關鍵商品**(攻略明列):朗錫爾道具店《隱身草》、瑪依拉村二樓《王者之劍》35000 金。
  → 對應 remake ITEM.DAT 解出的王者之劍價 35000(docs/22),互為佐證。

這些進度節點與道具,本專案用來:① 對齊 CTY→地名(配 Mike's RPG Center 世界位置);
② 設計 per-town 商店商品的難度遞進;③ 交叉驗證 RE 出的道具價格 / 進度旗標。

**為杜勝利這份心血留個見證——三十年後,它還在幫一個把這款未發售遊戲重做出來的專案帶路。**
