# 精訊版 DQ3 攻略來源致敬 + 地名對照(歷史見證)

這個反組譯 / remake 專案能把地名、進度、商品兜起來,靠的不只是 RE,也靠一整代玩家留下的攻略。
這份文件把用到的攻略來源列出來致敬,並整理出**地名英 / 中 / CTY 對照**(供 CTY→地名 收尾用)。

## 攻略作者致敬

- **杜勝利** — DQ3 攻略(收錄於 `vv0817.neocities.org/gametxt/09_dq3`)。從阿里阿罕開場到各城進度、
  地點、關鍵商品(如朗錫爾道具店的隱身草、瑪依拉村二樓 35000 金的王者之劍)逐一記下。
  這份攻略是本專案 CTY→地名、商店商品、進度串接的重要參照。**為這份心血留個見證。**
- **Dollar Cheng【孔方兄】= 陳煒元**、**青衫詩客 -- 小邱**、**Paul Shih**、**Jhon Horng-chinq** 等 —
  1994–95 BBS 討論串作者(見 [`dq3-bbs-1994.md`](dq3-bbs-1994.md)),攻略 + bug 修改 + 存檔佈局。
- **Mike's RPG Center**(`mikesrpgcenter.com/dw3`)— FC 版 Dragon Warrior III 世界地圖 + 英文地名,
  與精訊中文版同底,用來交叉確認各地點的世界位置與順序。

> 著作權:外部攻略只摘要要點、不轉貼全文;本文件為本反組譯專案的地理 / 進度交叉佐證之用。

阿里阿罕 → 雷貝(拿吉米之塔取盜賊鑰)→ 羅馬利亞 → 卡薩布 / 諾阿尼魯 / 香巴尼塔 →

> 此順序用於 remake 的 per-town 商店商品難度遞進(阿里阿罕最弱、往後漸強)。
