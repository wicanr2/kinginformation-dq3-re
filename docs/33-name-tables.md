# D3TXT00 全域名稱表(武器/道具/咒文/城鎮)

`D3TXT00.TXT` 是全域名稱字串表(非對白)。對白在 `D3TXT01..09.TXT`。

## 格式

- 開頭 u16 = 第 0 筆 offset = `offset 表位元組數`;記錄數 = `[0]/2`(本檔 = **760 筆**)。
- offset 表:`word[i]` = 第 i 筆相對檔頭 offset。
- 每筆 = **16-bit glyph 碼陣列 + `0xffff` 終結**;glyph 碼 × 32 = `D3TXT00.FON` 的 16×16 點陣 offset。
- 工具:`tools/render_names.py <lo> <hi>`(docker uv venv + PIL,輸出名稱點陣 PNG)。

## ルーラ 城鎮選單機制

- 選單(file 0x9249/0x5ef0):列已知城鎮 id → 名稱 = `id + 0x1ef(495)` → 渲染。
- 城鎮表 `DS 0x1fb`(20×7 byte,`id*7+0x1fb`)= **執行期填充**(訪問城鎮時寫座標),靜態檔為 0。
- ∴ 城鎮 id → 座標非靜態;**靜態座標記錄是 `cty_loc`(0x748,coord→CTY index,docs/32)**。
- 待解:CTY index ↔ 城鎮 id(20 名城)連結,以把名稱貼到 docs/32 地圖標記。
