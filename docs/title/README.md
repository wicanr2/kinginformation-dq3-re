# 標題 / 開場畫面

`FIRST.SCR` = 遊戲開場畫面,經反組譯解出格式後 render(`tools/title_render.py`):

- **640 × 350 像素、4bpp(16 色)**
- **row-interleaved planar**:每一掃描列 = plane0..3 各 80 byte(= 320 byte/列),350 列 × 320 = 112,000 byte
- 調色盤:目前取 `DQ3.PAL` 前 16 色(待與青衫先生攻略的標題畫面正確色彩核對)

`first_opening.png` = 還原結果:DQ3 雙主角開場原畫(鳥山明風格)。

`TIT*.P`(TITA~TITP、TIT0~3 等)為**壓縮**的同規格(640×350)標題/過場圖,共用表頭
`0a 05 01 01 00000000 | <u16 639><u16 349><u16 640><u16 350>`(邊界框),壓縮編碼待解。
